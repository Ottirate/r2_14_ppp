Un TP sur la génération de graphe

Un TP pour nous initier au classes.

Le sujet consisté à générer un graphe par rapport a des données donner.

Une fois le diagramme rédiger, il était temps de rédiger le code de chacune des classes.
Cette exercice, très peu complexe, ma laisser la liberté d'améliorer l'affichage du train dans le terminal, mais même aussi crée une fonction pour animer ce train.

Pour voir le code en entier, n'hésitez pas à aller sur répértoire git TP-SNCF.
